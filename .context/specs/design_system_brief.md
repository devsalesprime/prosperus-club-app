# 🎨 Resumo: Design System Prosperus

*Nota: Para especificações completas, a IA deve consultar o arquivo raiz `docs/DESIGN_SYSTEM.md`.*

- **Backgrounds:** `bg-prosperus-navy` (fundo principal), `bg-prosperus-card` (modais, cards, painéis).
- **Textos:** `text-prosperus-white` (primário), `text-prosperus-grey` (secundário).
- **Destaques:** `text-prosperus-gold` e `bg-prosperus-gold` para CTAs.
- **Modais Full-Screen (Aha Moments):** Usar `bg-black/95 backdrop-blur-md` e `z-[99999]`.
- **Ícones (Lucide):** `size={20} strokeWidth={1.5}` para ações.
